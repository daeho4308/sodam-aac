# 소담 AAC

<p align="center"><b>"소통을 담다"</b></p>

**소담 AAC**는 자폐 스펙트럼 아동을 비롯해 말로 의사소통이 어려운 아이들을 위한 **무료 보완대체의사소통(AAC) 앱**입니다.

그림 카드를 눌러 문장을 만들고, 기기가 아이 대신 말해줍니다.

## 주요 기능

- 🗣 **그림 카드 소통판** — 카드를 누르면 바로 말하고, 카드를 이어 문장을 만들 수 있어요
- 📡 **완전 오프라인** — 인터넷 없이 동작해요. 아이의 사용 기록이 기기 밖으로 나가지 않아요
- 🎙 **녹음 음성 카드** — 엄마·아빠가 직접 녹음한 목소리로 말하는 카드를 만들 수 있어요
- ⭐ **자주 쓰는 문장** — "화장실 가고 싶어요" 같은 문장을 저장해 두고 한 번에 말해요
- ✏️ **보호자 편집** — 사진·카메라·그림 상징 검색(ARASAAC)으로 우리 아이만의 카드를 만들어요
- 🔒 **화면 고정** — 아이가 다른 앱으로 나가지 못하게 잠글 수 있어요 (해제는 보호자 확인 필요)
- 📊 **사용 통계** — 자주 쓰는 낱말, 문장 길이 추이를 보호자·치료사가 확인할 수 있어요
- 💾 **백업/복원** — 소통판 전체를 파일 하나로 백업하고 다른 기기로 옮길 수 있어요

## 다운로드

👉 **[최신 버전 다운로드 (Releases)](../../releases/latest)**

| 파일 | 대상 |
|---|---|
| `sodam-…-arm64-v8a.apk` | 대부분의 Android 폰·태블릿 (2018년 이후 기기) |
| `sodam-…-armeabi-v7a.apk` | 구형 Android 기기 |

### Android 설치 방법

1. 태블릿(또는 폰)의 브라우저에서 위 다운로드 링크를 열어 `arm64-v8a.apk`를 받습니다.
2. 다운로드한 파일을 누르면 설치가 시작됩니다.
3. "출처를 알 수 없는 앱을 설치할 수 없음" 안내가 나오면 **설정 → 이 출처 허용**을 켠 뒤 다시 시도하세요.
4. 설치 후 첫 실행이 되면 바로 사용할 수 있습니다. 인터넷 연결이 없어도 됩니다.

> 💡 한국어 음성이 나오지 않으면: 기기 **설정 → 일반(또는 접근성) → 텍스트 음성 변환(TTS)** 에서 한국어 음성 데이터를 설치해 주세요. 삼성 기기는 삼성 TTS, 그 외 기기는 Google TTS를 권장합니다.

## 처음 시작하기 (보호자용)

1. **보호자 설정 들어가기** — 오른쪽 위 ⚙️ 버튼 → 곱셈 문제의 답을 숫자판으로 입력 (아이가 실수로 설정을 바꾸지 못하게 하기 위한 장치입니다)
2. **편집 모드 켜기** — 설정 첫 번째 스위치. 소통판으로 돌아가면 카드마다 연필 표시가 나타납니다
   - 카드를 누르면 수정, ＋로 추가, 길게 눌러 끌면 순서 변경
   - 낱말 입력 후 **[이미지 찾기]** 를 누르면 1만 개 이상의 그림 상징에서 검색해 넣을 수 있어요 (이때만 인터넷 필요)
   - **[목소리 녹음]** 으로 가족의 목소리를 카드에 담을 수 있어요
3. **아이에게 주기 전에** — 설정에서 **화면 고정 시작**을 누르면 아이가 홈 버튼으로 나갈 수 없습니다. 해제는 다시 보호자 확인을 거쳐 설정에서 합니다
4. **백업** — 설정 → 백업 내보내기. 만든 소통판을 파일로 저장해 Google Drive나 카카오톡에 보관하세요

## 자주 묻는 질문

**Q. 아이의 데이터가 수집되나요?**
아니요. 모든 데이터(소통판, 사진, 녹음, 사용 기록)는 기기 안에만 저장됩니다. 서버로 전송하지 않으며, 앱은 상징 검색 시에만 인터넷을 사용합니다.

**Q. 여러 기기에서 같은 소통판을 쓸 수 있나요?**
백업 내보내기 → 다른 기기에서 복원하면 됩니다 (사진·녹음 포함).
