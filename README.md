# 소담 AAC

<p align="center"><b>"소통을 담다"</b></p>

**소담 AAC**는 자폐 스펙트럼 아동을 비롯해 말로 의사소통이 어려운 아이들을 위한 **무료 보완대체의사소통(AAC) 앱**입니다.

그림 카드를 눌러 문장을 만들고, 기기가 아이 대신 말해줍니다.

📖 **[사용 설명서 보기](../../wiki)** — 설치부터 카드 만들기, 백업까지 화면과 함께 차근차근 설명해요

## 주요 기능

- 🗣 **그림 카드 소통판** — 카드를 누르면 바로 말하고, 카드를 이어 문장을 만들 수 있어요
- 📡 **완전 오프라인** — 인터넷 없이 동작해요. 아이의 사용 기록이 기기 밖으로 나가지 않아요
- 🎙 **녹음 음성 카드** — 엄마·아빠가 직접 녹음한 목소리로 말하는 카드를 만들 수 있어요
- ⭐ **자주 쓰는 문장** — "화장실 가고 싶어요" 같은 문장을 저장해 두고 한 번에 말해요
- ✏️ **보호자 편집** — 사진·카메라·그림 상징 검색(ARASAAC)으로 우리 아이만의 카드를 만들어요
- 🔒 **화면 고정** — 아이가 다른 앱으로 나가지 못하게 잠글 수 있어요 (해제는 보호자 확인 필요)
- 📊 **사용 통계** — 자주 쓰는 낱말, 문장 길이 추이를 보호자·치료사가 확인할 수 있어요
- 💾 **백업/복원** — 소통판 전체를 파일 하나로 백업하고 다른 기기로 옮길 수 있어요

### Android 설치 방법

> 이 앱은 Play 스토어가 아닌 이 페이지에서 직접 내려받아 설치합니다.
> 처음 한 번만 "허용" 과정을 거치면 되고, 이후 업데이트는 앱 안에서 할 수 있어요.
> **아이가 쓸 그 기기(태블릿·폰)에서 직접** 아래 순서를 진행하세요.

**1단계 · 앱 파일 내려받기**

아이가 사용할 기기의 인터넷 브라우저(크롬 등)에서 아래 주소를 엽니다.

```
github.com/daeho4308/sodam-aac/releases/latest
```

화면을 아래로 내리면 **Assets**(또는 "에셋") 목록이 있습니다.
`sodam-…-arm64-v8a.apk` 를 눌러 내려받으세요. 크기는 약 20MB입니다.

- "이런 종류의 파일은 기기에 위험을 줄 수 있습니다"라는 경고가 나오면 **[계속]** 또는 **[확인]** 을 누릅니다. (Play 스토어를 거치지 않는 모든 앱에 표시되는 일반적인 안내입니다)
- 다운로드가 끝나면 알림창이나 **파일 앱 → 다운로드** 폴더에서 받은 파일을 확인할 수 있습니다.

**2단계 · 설치 시작하기**

받은 `sodam-…apk` 파일을 누릅니다. 설치 화면이 열리면 **[설치]** 를 누르세요.

**3단계 · "알 수 없는 앱 설치" 허용 (처음 한 번만)**

"보안상의 이유로 이 출처의 알 수 없는 앱을 설치할 수 없습니다"라는 안내가 나오는 경우가 있습니다. 정상이며, 아래처럼 허용하면 됩니다.

1. 안내 창의 **[설정]** 을 누릅니다.
2. **"이 소스에서 가져온 앱 설치 허용"** 스위치를 켭니다.
3. 뒤로 가기를 눌러 돌아온 뒤 **[설치]** 를 다시 누릅니다.

> 제조사별 화면 이름이 조금씩 다릅니다.
> **삼성**: 설정 → 보안 및 개인 정보 보호 → 기타 보안 설정 → 출처를 알 수 없는 앱 설치
> **LG·기타**: 설정 → 앱 → 특별한 앱 접근 → 알 수 없는 앱 설치
> 여기서 방금 파일을 연 앱(크롬, 내 파일 등)을 찾아 허용해 주면 됩니다.

**4단계 · 실행하기**

설치가 끝나면 **[열기]** 를 누르거나, 앱 목록에서 초록색 말풍선 아이콘의 **소담 AAC**를 실행합니다.
회원가입이나 로그인 없이 바로 사용할 수 있고, 인터넷 연결도 필요 없습니다.

**5단계 · 한국어 음성 확인 (소리가 안 날 때만)**

카드를 눌렀는데 소리가 나지 않으면 기기에 한국어 음성 데이터가 없는 경우입니다.

기기 **설정 → 일반(또는 접근성) → 텍스트 음성 변환(TTS) 출력** 에서 한국어 음성을 설치해 주세요.
삼성 기기는 "삼성 TTS", 그 외 기기는 "Google 텍스트 음성 변환"을 권장합니다.
설치 후 소담 AAC의 **설정 → 음성 미리 듣기**로 확인할 수 있습니다.

### 업데이트 방법

새 버전이 나오면 앱 안에서 바로 받을 수 있습니다.

앱의 ⚙️ 버튼 → 보호자 확인(곱셈 문제) → 맨 아래 **업데이트** 항목을 확인하세요.
새 버전이 있으면 "새 버전 v0.0.0 업데이트"가 표시되고, 누르면 자동으로 내려받아 설치를 시작합니다.
**업데이트해도 만들어 둔 카드·사진·녹음·설정은 그대로 유지됩니다.**

## 처음 시작하기 (보호자용)

1. **보호자 설정 들어가기** — 오른쪽 위 ⚙️ 버튼 → 곱셈 문제의 답을 숫자판으로 입력 (아이가 실수로 설정을 바꾸지 못하게 하기 위한 장치입니다)
2. **편집 모드 켜기** — 설정 첫 번째 스위치. 소통판으로 돌아가면 카드마다 연필 표시가 나타납니다
   - 카드를 누르면 수정, ＋로 추가, 길게 눌러 끌면 순서 변경
   - 낱말 입력 후 **[이미지 찾기]** 를 누르면 1만 개 이상의 그림 상징에서 검색해 넣을 수 있어요 (이때만 인터넷 필요)
   - **[목소리 녹음]** 으로 가족의 목소리를 카드에 담을 수 있어요
3. **아이에게 주기 전에** — 설정에서 **화면 고정 시작**을 누르면 아이가 홈 버튼으로 나갈 수 없습니다. 해제는 다시 보호자 확인을 거쳐 설정에서 합니다
4. **백업** — 설정 → 백업 내보내기. 만든 소통판을 파일로 저장해 Google Drive나 카카오톡에 보관하세요

> 더 자세한 사용법은 📖 **[사용 설명서](../../wiki)** 에 화면과 함께 정리되어 있어요.

## 자주 묻는 질문

**Q. 아이의 데이터가 수집되나요?**
아니요. 모든 데이터(소통판, 사진, 녹음, 사용 기록)는 기기 안에만 저장됩니다. 서버로 전송하지 않으며, 앱은 상징 검색 시에만 인터넷을 사용합니다.

**Q. 여러 기기에서 같은 소통판을 쓸 수 있나요?**
백업 내보내기 → 다른 기기에서 복원하면 됩니다 (사진·녹음 포함).
